## Color Scheme

### Primary Colors:

- **Orange-Red Gradient**: Headers (#f97316 â†’ #f43f5e)
- **White**: Card backgrounds (#ffffff)
- **Neutral Gray**: Text (#404040)

### Action Colors:

- **Blue**: Save, Email buttons (#3b82f6)
- **Green**: Login, Call buttons (#22c55e)
- **Red**: Logout button (#ef4444)
- **Orange-Red**: Submit button (#f97316 â†’ #f43f5e)
- **Purple**: Language selection (#a855f7)

### Status Colors:

- **Green**: Success, Active, Logged in (#22c55e)
- **Red**: Emergency, Alerts (#ef4444)
- **Blue**: Information, Upcoming (#3b82f6)
- **Orange**: Pending tasks (#fb923c)

## Typography

### Font Family:

```
'Times New Roman', Times, serif
```

### Font Sizes (Elderly-Friendly):

- **Headings H1**: 2.2rem (35.2px)
- **Headings H2**: 1.8rem (28.8px)
- **Headings H3**: 1.4rem (22.4px)
- **Body Text**: 1.2rem (19.2px)
- **Small Text**: 1rem (16px)

### Font Weights:

- **Extra Bold**: 800 (main headings)
- **Bold**: 700 (section headers, labels)
- **Semi-Bold**: 600 (input text, body)

## Spacing System

### Padding:

- **Page**: 6px (px-6 py-6)
- **Sections**: 6px (p-6)
- **Items**: 4px (p-4)
- **Buttons**: 5px horizontal (px-5)

### Gaps:

- **Section spacing**: 6 (space-y-6)
- **Item spacing**: 3-5 (space-y-3 to space-y-5)
- **Icon-text gaps**: 3 (gap-3)

### Borders:

- **Border width**: 3px (border-3)
- **Border radius**:
  - Small: 8px (rounded-lg)
  - Medium: 12px (rounded-xl)
  - Large: 16px (rounded-2xl)

## Interactive Elements

### Button Sizes:

- **Large buttons**: h-14 (56px height)
- **Icon buttons**: w-16 h-16 (64px Ã— 64px)
- **Touch target**: Minimum 48px for accessibility

### Hover Effects:

```css
hover:scale-105      /* Slight zoom on large buttons */
hover:scale-110      /* More zoom on small buttons */
hover:shadow-lg      /* Elevation increase */
hover:bg-darker      /* Color darkening */
```

### Icons:

- **Header icons**: w-8 h-8 (32px)
- **List icons**: w-7 h-7 (28px)
- **Button icons**: w-5 h-5 to w-6 h-6 (20-24px)
- **Stroke width**: 3 (thick for visibility)

## Animation Timings

### Transitions:

- **Settings slide-in**: 300ms ease-out
- **Button hover**: 200ms
- **Card hover**: 300ms
- **Toast notifications**: 400ms
